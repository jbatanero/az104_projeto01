# Gerenciamento de Máquinas Virtuais no Microsoft Azure

Descrição

Este repositório contém anotações, resumos e dicas práticas sobre o gerenciamento de máquinas virtuais no Microsoft Azure. Ele foi criado como parte do desafio da plataforma DIO, com o objetivo de consolidar o aprendizado e servir como referência para estudos futuros.

Objetivos de Aprendizagem

- Aplicar os conceitos aprendidos sobre Azure em um ambiente prático.
- Documentar os processos de criação e gerenciamento de VMs.
- Utilizar o GitHub como repositório de documentação técnica.
